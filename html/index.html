About

Playgrub is an easy way to make playlists from web pages and play them in Playdar.

To install Playgrub drag this link to your browser's bookmarks bar:

playgrub bookmarklet

Playgrub is open source. Get the code from Github.

Supported Sites

Using the bookmarklet on any of the following sites will create a playlist out of the listed songs.

Last.fm
The Y! Radish
Musicbraniz Release Pages
BBC Radio 1 Programmes
Grooveshark Playlist Export Page
FIQL Playlists

Playdar Players

Playgrub gives you the option of playback in Playlick and Spiffdar
